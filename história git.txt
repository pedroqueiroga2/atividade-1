O que é GIT?

O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 
por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux. 
Um número impressionante de projetos de software depende do Git para controle de versão, 
incluindo projetos comerciais e de código-fonte aberto.

como surigu o git: 

Inicialmente, as pessoas desenvolvedoras que trabalhavam no kernel (núcleo) 
do Linux usavam o sistema do BitKeeper. 
Porém, os criadores removeram o acesso gratuito após uma polêmica sobre direitos autorais. 
Insatisfeito, Torvalds decidiu criar um sistema semelhante com código aberto, dando origem
assim ao Git.

Propósito:

GIT usa um modelo de dados que ajuda a garantir a integridade criptográfica de qualquer coisa 
presente dentro de um repositório. 
Cada vez que um arquivo é adicionado ou um commit é feito, suas somas de verificação são 
geradas. Da mesma forma, eles são recuperados através de suas somas de verificação também.

Diferença de GIT e GITHUB:

O Github é uma “rede social dev” em que é possível armazenar e compartilhar projetos de 
desenvolvimento de software. O Git é um sistema de controle de versão de arquivos;
